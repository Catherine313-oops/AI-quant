山东黄金 (600547.SH) 涨跌分类模型¶

项目概述¶

  • 数据来源：Tushare Pro — 山东黄金(600547.SH) 近10年日线数据
  • 任务目标：二分类 — 预测下一交易日收盘价涨跌
    • 涨(1)：当日收盘价 ≤ 次日收盘价 (close[t] < close[t+1])
    • 跌(0)：当日收盘价 > 次日收盘价 (close[t] >= close[t+1])
  • 数据集划分：
    • 训练集：2016-01-01 ～ 2023-12-31（8年）
    • 测试集：2024-01-01 ～ 2025-12-31（2年）
  • 模型选择：逻辑回归、决策树、随机森林
  • 特征类型：仅使用基础衍生特征（收益率、波动幅度、成交量变化、滚动统计、日历特征）

3. 构造标签 (Label)¶

规则：

  • Label[i] = 1：第 i+1 日收盘价 > 第 i 日收盘价（涨）
  • Label[i] = 0：第 i+1 日收盘价 ≤ 第 i 日收盘价（跌）

使用 shift(-1) 将未来一天的涨跌信息对齐到当天。

In [3]:
# 构造标签：下一日收盘价 > 今日收盘价 → 1（涨），否则 → 0（跌）
df['Label'] = (df['Close'].shift(-1) > df['Close']).astype(int)

# 统计分布
label_counts = df['Label'].value_counts()
print(f"\U0001f4ca 标签分布:")
print(f"   涨 (1): {label_counts.get(1, 0)} 条 ({df["Label"].mean():.2%})")
print(f"   跌 (0): {label_counts.get(0, 0)} 条 ({1 - df["Label"].mean():.2%})")
print(f"   总样本: {len(df)} 条（含最后一行 NaN，后续会被 dropna 去除）")

4. 特征工程¶

构建 13 个基础特征，分为 5 组：

分组 特征 说明
(A) 滞后收益率 ret_1d, ret_2d, ret_3d, ret_5d 过去1/2/3/5日收益率
(B) 价格波动 range_pct, intraday_chg 日内波幅占比、日内涨跌幅
(C) 成交量 volume_chg, volume_ratio_5d 成交量变化率、量比(5日均量)
(D) 滚动统计 close_ma5, close_std5, close_vs_ma5 5日均线、标准差、偏离度
(E) 日历特征 month, dayofweek 月份(季节性)、星期(周内效应)
In [4]:
# --- (A) 滞后收益率 ---
df['ret_1d'] = df['Close'].pct_change(1)
df['ret_2d'] = df['Close'].pct_change(2)
df['ret_3d'] = df['Close'].pct_change(3)
df['ret_5d'] = df['Close'].pct_change(5)

# --- (B) 价格波动特征 ---
df['range_pct'] = (df['High'] - df['Low']) / df['Close']
df['intraday_chg'] = (df['Close'] - df['Open']) / df['Open']

# --- (C) 成交量特征 ---
df['volume_chg'] = df['Volume'].pct_change(1)
df['volume_ratio_5d'] = df['Volume'] / df['Volume'].rolling(5).mean()

# --- (D) 滚动统计特征 ---
df['close_ma5'] = df['Close'].rolling(5).mean()
df['close_std5'] = df['Close'].rolling(5).std()
df['close_vs_ma5'] = (df['Close'] - df['close_ma5']) / df['close_ma5']

# --- (E) 日历特征 ---
df['month'] = df['trade_date'].dt.month
df['dayofweek'] = df['trade_date'].dt.dayofweek

feature_cols = [
    'ret_1d', 'ret_2d', 'ret_3d', 'ret_5d',
    'range_pct', 'intraday_chg',
    'volume_chg', 'volume_ratio_5d',
    'close_ma5', 'close_std5', 'close_vs_ma5',
    'month', 'dayofweek'
]
print(f"\u2705 特征工程完成，共构建 {len(feature_cols)} 个特征:")
for i, col in enumerate(feature_cols, 1):
    print(f"   {i:2d}. {col}")

# 定义模型
models = {
    "Logistic Regression": LogisticRegression(
        C=1.0, max_iter=1000, random_state=42
    ),
    "Decision Tree": DecisionTreeClassifier(
        max_depth=5, min_samples_leaf=10, random_state=42
    ),
    "Random Forest": RandomForestClassifier(
        n_estimators=100, max_depth=8, min_samples_leaf=5,
        random_state=42, n_jobs=-1
    )
}

# 训练 & 预测
results = {}
for name, model in models.items():
    if name == "Logistic Regression":
        model.fit(X_train_scaled, y_train)
        y_pred = model.predict(X_test_scaled)
        y_prob = model.predict_proba(X_test_scaled)[:, 1]
    else:
        model.fit(X_train, y_train)
        y_pred = model.predict(X_test)
        y_prob = model.predict_proba(X_test)[:, 1]

    results[name] = {"y_pred": y_pred, "y_prob": y_prob, "model": model}
    print(f"\u2705 {name} 训练完成")

指标 说明
Accuracy 整体准确率
Precision 预测为涨的样本中，实际为涨的比例
Recall 实际为涨的样本中，被正确预测的比例
F1-score Precision 与 Recall 的调和平均
ROC AUC 模型区分能力（基于预测概率）
